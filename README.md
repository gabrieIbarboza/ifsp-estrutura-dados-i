# Data Structures I - IFSP

This repository contains materials and code for the Data Structures I course at the Federal Institute of SÃ£o Paulo (IFSP). The course covers fundamental concepts and implementations of various data structures in C programming language.

## Course Contents

The repository is organized into several modules, each focusing on different aspects of data structures and algorithms:

1. **C Language Review**
2. **Functions**
3. **Strings**
4. **Structs**
5. **Pointers**
6. **File Manipulation**
7. **Memory Allocation**
8. **Recursive Functions**
9. **Abstract Data Types**
10. **Static Lists**
11. **Linked Lists**
12. **Dynamic Stacks**
13. **Doubly Linked Lists**
14. **Search and Sort Algorithms for Vectors**
15. **Algorithm Analysis**
16. **Hash Tables**
17. **Binary Trees**
18. **AVL Trees**
19. **Red-Black Trees**
